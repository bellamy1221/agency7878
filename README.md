# TSBLV

Портфолио независимой digital-студии TSBLV.

## Стек

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- Motion (`motion/react`)
- Phosphor Icons

## Команды

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Контент

Проекты и кейсы: `src/data/projects.ts`  
Контакты и навигация: `src/data/site.ts`

Все кейсы помечены как независимые концепции и легко заменяются реальными работами.
